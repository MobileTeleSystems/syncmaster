# Архитектура { #reference-architecture }

## Компоненты

SyncMaster содержит следующие компоненты:

- [Фронтенд][frontend], основной пользовательский интерфейс.
- [Сервер REST API][server], предоставление REST API для извлечения и манипулирования сущностями.
- [Воркер(рабочий процесс)][worker], выполнение фактических работ по передаче (процессы ETL).
- [Планировщик][scheduler], планирование трансферов, которые будут выполнены в будущем.
- [Реляционная база данных][database] для хранения внутренних данных.
- [Брокер сообщений][message-broker] для коммуникации между  Сервером/Планировщиком и Воркером(рабочим процессом).
