# 0.2.2 (2025-04-11) { #0.2.2 }

## Неисправность

- Используйте `PUT /v1/qroups/:id` вместо `PATCH /v1/qroups/:id`.
- Используйте `PUT /v1/qroups/:id/users/:id` вместо `PATCH /v1/qroups/:id/users/:id`.
- Используйте `PUT /v1/queues/:id` вместо `PATCH /v1/queues/:id`.
- Теперь допустимая длина имён должна быть в диапазоне от 3 до 128 символов, а не от 1 до бесконечности.

## Улучшения

- Теперь имя очереди может содержать любые печатные символы ASCII.
- Имя очереди всегда пишется в нижнем регистре. Пробелы, дефисы и подчёркивания заменены символом `-`.

## Исправления ошибок

- Вызов `kinit` перед началом сеанса Spark, подключающегося к кластеру `Hive`. ([#225](https://github.com/MobileTeleSystems/syncmaster/issues/225))
- Исправлена ​​ошибка, из-за которой подключение `HDFS` пыталось использовать анонимную аутентификацию вместо имени пользователя/пароля. ([#225](https://github.com/MobileTeleSystems/syncmaster/issues/225))
- Исправлена ​​ошибка, из-за которой при обновлении очереди игнорировалось имя и не сбрасывалось описание.
