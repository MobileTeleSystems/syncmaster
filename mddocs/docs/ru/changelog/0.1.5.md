# 0.1.5 (2024-04-22) { #0.1.5 }

## Критичные изменения

- Передайте текущий `Run` в `CREATE_SPARK_SESSION_FUNCTION`. Это позволяет использовать информацию о запуске/передаче/группе для параметров сеанса Spark,
например, `appName` или пользовательских. ([#38](https://github.com/MobileTeleSystems/syncmaster/issues/38))

## Улучшения

- Уменьшен размер образа бэкенда ([#44](https://github.com/MobileTeleSystems/syncmaster/issues/44))

## Исправления ошибок

- Исправлена ​​ошибка 500 при запросе `PATCH v1/connections/:id` с переданным значением поля `auth_data.password` ([#39](https://github.com/MobileTeleSystems/syncmaster/issues/39))
- Не используйте `asyncio.gather` с запросами SQLAlchemy ([#40](https://github.com/MobileTeleSystems/syncmaster/issues/40))
- Исправлена ​​ошибка 500 при создании HDFS-подключения ([#41](https://github.com/MobileTeleSystems/syncmaster/issues/41))
- Исправлено отсутствие поля `options` в параметрах передачи с типами `hdfs` и `s3`
